# Curator

**Memory that keeps up with you.**

Curator is a local, encrypted evolutionary memory layer for AI agents. It preserves project decisions, tracks what later replaced them, and helps a returning agent resume with the standards you hold now rather than the context you left behind.

## Release status

Curator is a working product entering IP hardening, licensing, and public packaging preparation.

This repository is the official release and product-information channel. It does not currently contain distributable software.

## What it changes

- **Resume with the delta:** Surface what changed since a project was last active.
- **Domain-scoped memory:** Keep projects, personal context, and organizational knowledge separated.
- **Evolution instead of accumulation:** Superseded or contradicted decisions do not remain equal to current ones.
- **Explainable recall:** Returned context includes reasons, lineage, and feedback history.
- **Local and encrypted:** Memory remains under operator control and is encrypted at rest.
- **MCP-native:** Designed to serve compatible AI clients through focused memory tools.
